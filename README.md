# Bank Api Python

This is a personal practice project for creating a bank API using Flask and MongoDB.

## Features

- Create account
- Login account
- Retrieve user information by ID
- Retrieve account balance by user ID

## Technologies Used

- Python
- Flask
- MongoDB
- Bcrypt
- JWT (JsonWebToken)
